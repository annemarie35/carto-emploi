    Recherche d'une offre

    • Comment utiliser cet écran ?

    A quoi sert cet écran ?Haut de page

    • Cet écran vous présente la description de l’offre d’emploi que vous avez sélectionnée.
    • Si elle vous intéresse, vous pouvez y répondre selon les indications et modalités décrites au bas de l’offre. Vous pouvez également l’imprimer, l’enregistrer (en l’ajoutant à votre classeur) ou l’envoyer à un(e) ami(e).

    Comment utiliser cet écran ?Haut de page

    • Pour accéder à l’offre située juste avant dans la liste, cliquez sur le lien Offre précédente.
    • Pour accéder à l’offre située juste après dans la liste, cliquez sur le lien Offre suivante.
    • Sur certaines offres, vous pouvez cliquer sur le lien Compétences spécifiques qui vous permet de consulter les compétences spécifiques demandées par l’employeur pour le poste proposé.

    • Pour répondre à l’offre, suivez les indications et les modalités précisées au bas de l’offre : les coordonnées et le moyen de contact y sont précisés.

    Sur certaines offres, un bouton Postuler vous permet de « télécandidater » en ligne à l’offre d’emploi (envoi automatique d’un formulaire au conseiller Pôle Emploi).
    Certaines offres sont proposées par nos partenaires, pour y postuler il vous suffit de cliquer sur le logo du partenaire proposant l’offre. Vous serez alors redirigé vers son site internet dans lequel vous pourrez postuler à l’offre en question.

    • Si vous souhaitez conserver une offre d’emploi dans votre classeur d’offres (pour la consulter ultérieurement), cliquez sur le bouton Ajouter à mon classeur (une fois l’offre enregistrée dans votre classeur, le bouton devient non cliquable).
    A noter : l’ajout de l’offre dans votre classeur nécessite de créer votre espace personnel ou de vous identifier si vous en avez déjà un.

    • Si vous souhaitez envoyer cette offre par e-mail à une personne de votre connaissance (dont vous connaissez l’adresse e-mail), cliquez sur le bouton Envoyer à un ami.

    • Si vous souhaitez réutiliser les critères définis dans cette recherche pour éviter de les ressaisir à chaque nouvelle consultation ou afin de recevoir les offres d’emploi automatiquement par e-mail (abonnement aux offres), cliquez sur le bouton Enregistrer les critères(une fois les critères enregistrés, le bouton devient non cliquable).
    A noter : l’enregistrement des critères de recherche nécessite de créer votre espace personnel ou de vous identifier si vous en avez déjà un.

    • Les critères de votre recherche vous sont rappelés en cliquant sur Afficher les critères : si vous souhaitez les modifier, cliquez sur le bouton Modifier les critères.

    • Pour démarrer une nouvelle recherche avec d’autres critères que ceux en cours, cliquez sur le bouton Nouvelle recherche.

    • Pour imprimer la page, cliquez sur le bouton version imprimable en haut de page.

    • Pour revenir à la page précédente, cliquez sur le bouton Retour aux résultats.

Développeur / Développeuse web

Autres offres pour le métier : Développeur / Développeuse web

Description de l'offre

Intégré(e) au sein d'une équipe d'une trentaine de personnes, organisée en projets et avec une gestion agile, vous principales missions sont:
- Développer et maintenir des applications web de gestion et traitements de contenu principalement développées en JAVA (JEE/GWT) et Javascript
- Assurez une veille technologique constante pour rester au plus haut niveau et garantir une adaptation des applications existantes reflétant l'état de l'art du domaine

De formation ingénieur informatique ou équivalent (Bac+5), vous justifiez d'une expérience significative d'au moins 3 ans sur un poste similaire. .

Compétences techniques requises
- JEE, GWT, JavaScript (JQuery, AngularJS, OpenLayer,) HTML, CSS
- Maîtrise de Linux

Aptitudes personnelles
Rigueur, autonomie, méthode, dynamisme, capacité d'analyse et de et de travail en équipe qui vous permettront de vous adapter rapidement à un environnement technique.
